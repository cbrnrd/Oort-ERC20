WHAT IS OORT?

OORT is a decentralized key-value store built on Ethereum. It uses its own token (OORT) for purchasing storage. Any data you write to the Oort Cloud is tied to your Ethereum address and is yours forever.

Easy to use
Oort is, in its nature, simple. It only has two functions and does all the rest for you.
Secure
All data you store is yours and only yours. You are the owner of your data, nobody else can edit, delete, or add to your collection of data.
Trusted
Oort and its token both run on the Ethereum network (what's this?), so it will never have downtime and is completely decentralized.
